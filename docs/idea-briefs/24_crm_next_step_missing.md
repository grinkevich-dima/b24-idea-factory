# Идея 24 — CRM: Сделки без следующего шага

## 1) Коротко (одна фраза)
Приложение находит сделки, где после контакта не запланировано следующее дело, и помогает менеджеру не терять продолжение коммуникации.

## 2) Для кого (ICP)
- РОП
- Тимлид продаж
- Владелец CRM‑регламента

## 3) Боль и JTBD
- Менеджер поговорил с клиентом, но не поставил следующий шаг.
- Сделка выглядит активной, хотя по ней нет будущего действия.

## 4) Измеримый эффект
- Снижение доли сделок без следующего действия.
- Рост дисциплины follow‑up и конверсии в следующий этап.

## 5) УТП (позиционирование)
**“Покажем сделки без следующего шага и поможем поставить дело в один клик.”**

## 6) MVP (3–5 фич)
1. **Список активных сделок без будущего дела.**
2. **Фильтры по воронке, стадии и ответственному.**
3. **Быстрое создание активности из списка.**
4. **Уведомление ответственному и дайджест РОПу.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Сделки, активности/дела, стадии, ответственные, даты завершения и будущих дел.

### Триггеры и обновление
- Проверка после изменения сделки/дела и периодический контроль активных сделок.

### Права и безопасность
- Не читаем содержимое коммуникаций; используем наличие и даты активностей.

### Производительность
- Проверяем активные сделки за выбранный период и кэшируем список нарушений.

## 8) Риски (и как снизить)
- **Дублирование штатных напоминаний** → фокус на сделках без любого следующего шага.
- **Разные регламенты по стадиям** → правила по воронкам.
- **Много старых сделок** → топ‑список и фильтры.

## 9) Метрики успеха
- Доля сделок без будущего дела.
- Количество созданных следующих шагов.
- Время от контакта до постановки следующего дела.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “следующий шаг CRM”
- “сделки без дел”
- “follow up CRM”
- “контроль сделок”
- “активности в сделках”

## 11) Упаковка для Маркета (черновик)
### Название
Сделки без следующего шага

### Буллеты ценности
- Находит сделки без будущего действия
- Быстрое создание активности
- Фильтры по воронкам и стадиям
- Дайджест РОПу
