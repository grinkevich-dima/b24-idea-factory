# Идея 16 — Задачи: План/факт часов по задачам

## 1) Коротко (одна фраза)
Простой отчёт сравнивает плановые и фактические трудозатраты по задачам, проектам и исполнителям.

## 2) Для кого (ICP)
- Руководитель проекта
- Руководитель сервисной команды
- Финансовый/операционный менеджер

## 3) Боль и JTBD
- Команда тратит больше времени, чем планировалось, но это видно слишком поздно.
- План/факт собирается вручную в таблицах.

## 4) Измеримый эффект
- Снижение перерасхода часов по проектам.
- Более точная оценка будущих задач.

## 5) УТП (позиционирование)
**“План/факт часов по задачам в одном отчёте без внешних таблиц.”**

## 6) MVP (3–5 фич)
1. **Отчёт план/факт по задачам, проектам и исполнителям.**
2. **Фильтры по периоду, группе и ответственному.**
3. **Подсветка задач с превышением плановых часов.**
4. **Экспорт отчёта или ссылка для руководителя.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Задачи, учёт времени, ответственные, группы/проекты, плановые оценки если используются.

### Триггеры и обновление
- Обновление по запросу и ночной пересчёт агрегатов.

### Права и безопасность
- Читаем только поля задач и данные учёта времени; не храним содержимое комментариев.

### Производительность
- Ограничение периодом отчёта и кэш агрегатов по задачам.

## 8) Риски (и как снизить)
- **Не все ведут учёт времени** → показывать покрытие учёта как отдельную метрику.
- **Плановые часы могут храниться в разных полях** → настройка поля плана.
- **Сложность биллинга** → MVP не считает деньги, только часы.

## 9) Метрики успеха
- Процент задач с учётом времени.
- Отклонение факт/план по часам.
- Количество задач с превышением порога.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “план факт задач”
- “учёт времени”
- “трудозатраты”
- “отчёт по часам”
- “задачи Bitrix24”

## 11) Упаковка для Маркета (черновик)
### Название
План/факт часов по задачам

### Буллеты ценности
- Видите перерасход часов
- Фильтры по проектам и людям
- Подсветка превышений
- Поддержка быстрых управленческих отчётов
