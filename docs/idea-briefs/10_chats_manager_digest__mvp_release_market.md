# 10 — Дайджест для руководителя: MVP + релиз + текст Маркета

## УТП

**Ежедневный дайджест «что горит» в рабочий чат — просрочки, зависшие задачи и сделки без движения в одном коротком сообщении. Не статистика, а призыв к действию.**

## MVP (3–5 фич)

1. **Формирование дайджеста** — агрегация: просроченные задачи (топ-5), сделки без активности > N дней (топ-5), задачи без дедлайна (счётчик).
2. **Доставка в чат** — отправка форматированного сообщения в личный чат или групповой чат отдела по расписанию.
3. **Расписание** — настройка времени отправки (по умолчанию: 09:00 пн–пт) и получателей.
4. **Настройка контента** — что включать в дайджест (задачи / сделки / оба), порог «дней без активности», отделы/проекты.
5. **Умная тишина** — если нечего сообщить (всё в порядке), дайджест не отправляется.

## Не-MVP (отложено)

- Дайджест на email (помимо чата)
- Выбор конкретных метрик/KPI для включения
- Интерактивные кнопки в сообщении (переназначить, продлить)
- История дайджестов (архив)
- Сравнение «сегодня vs вчера»

## План релиза

### Неделя 1
- День 1: API-запросы: просроченные задачи + сделки без активности
- День 2–3: Логика формирования дайджеста (топ-N + форматирование)
- День 4: Отправка сообщения в чат (API im.message.add)
- День 5: Настройки (расписание, получатели, контент)

### (Бонус: День 6–7, если есть время)
- Тестирование + edge cases (пустой дайджест, большой портал)
- Публикация + тексты

## Метрики успеха

| Метрика | Цель (3 мес.) |
|---------|---------------|
| Установки | 600+ |
| Активные порталы (30 дней) | 250+ |
| Средняя оценка | ≥ 4.5 |
| Retention 30 дней | ≥ 70% |
| Открытие дайджеста (клик по ссылке в сообщении) | ≥ 50% |

## Риски и смягчения

| Риск | Смягчение |
|------|-----------|
| Шум (слишком длинный дайджест) | Жёсткий лимит: топ-5 задач + топ-5 сделок. Остальное — «и ещё N...» |
| Конкуренция с PALAX | Дифференциация: «что горит» vs «что сделано». Фокус на проблемах, не статистике |
| Права на чат | Запрашиваем только отправку (im.message.add), не полный доступ к чатам |
| Временные зоны | Использовать timezone портала для расписания |

## Черновик текста для Маркета

### Заголовок
**Утренний дайджест — «что горит» в чат каждое утро**

### Подзаголовок
Просрочки, зависшие задачи, сделки без движения — топ-10 проблем в одном сообщении. Не статистика, а план действий на утро.

### Буллеты ценности
- ✅ Каждое утро в 09:00 — краткий список «где горит» прямо в чат
- ✅ Кросс-модульно: задачи + сделки + дела в одном сообщении
- ✅ Если всё ок — дайджест не приходит (нет спама)
- ✅ Настройка за 1 минуту: выбери отдел, время, что включать
- ✅ Минимальные права — только чтение + отправка сообщений

### Quick Start
1. Установите приложение
2. Выберите получателя (вы или групповой чат)
3. Настройте время (по умолчанию: 09:00)
4. Завтра утром получите первый дайджест

### FAQ
- **Чем отличается от «Ежедневного отчёта» (PALAX)?** — PALAX показывает статистику (сколько звонков, сколько задач закрыто). Мы показываем проблемы (что просрочено, где нет активности, что требует решения).
- **Что если каждый день 100 просрочек?** — Показываем топ-5 по критичности + «и ещё 95...» со ссылкой на полный список.
- **Можно ли отправлять не каждый день?** — Да: ежедневно, 3 раза в неделю (пн/ср/пт), или еженедельно (пн).
