# Идея 26 — CRM: Встречи/звонки без результата в CRM

## 1) Коротко (одна фраза)
Приложение находит завершённые звонки и встречи без результата или следующего действия, чтобы CRM не превращалась в пустой журнал активностей.

## 2) Для кого (ICP)
- РОП
- Тимлид продаж
- Владелец CRM‑регламента

## 3) Боль и JTBD
- Менеджеры закрывают активности без понятного итога.
- После встречи нет результата, следующего шага и причины, почему сделка двигается или стоит.

## 4) Измеримый эффект
- Рост доли активностей с результатом и следующим шагом.
- Более качественная история работы со сделкой.

## 5) УТП (позиционирование)
**“Покажем активности без результата и поможем быстро довести их до регламента.”**

## 6) MVP (3–5 фич)
1. **Список завершённых активностей без результата/комментария/следующего дела.**
2. **Правила по типам активностей и стадиям.**
3. **Шаблоны быстрых результатов.**
4. **Отчёт по менеджерам и воронкам.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- CRM‑активности, сделки/лиды, тип активности, ответственные, будущие дела.

### Триггеры и обновление
- Проверка после завершения активности и периодический отчёт за период.

### Права и безопасность
- Не анализируем тексты звонков или писем; проверяем наличие результата и метаданные активности.

### Производительность
- Фильтры по завершённым активностям за последние 7/30 дней.

## 8) Риски (и как снизить)
- **Разные правила для звонков и встреч** → настройки по типам.
- **Менеджеры могут писать формальные результаты** → MVP не оценивает качество текста.
- **Шум при старых активностях** → проверка только новых после включения правила.

## 9) Метрики успеха
- Доля активностей без результата.
- Количество исправленных активностей.
- Доля активностей со следующим шагом.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “результат звонка”
- “результат встречи”
- “CRM активности”
- “следующее дело”
- “контроль менеджеров”

## 11) Упаковка для Маркета (черновик)
### Название
Контроль результатов звонков и встреч

### Буллеты ценности
- Находит активности без результата
- Шаблоны быстрых итогов
- Контроль следующего действия
- Отчёт по менеджерам
