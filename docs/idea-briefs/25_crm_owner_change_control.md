# Идея 25 — CRM: Контроль смены ответственного в CRM

## 1) Коротко (одна фраза)
Журнал и контроль передачи сделок показывают, что произошло после смены ответственного и не потерялась ли сделка.

## 2) Для кого (ICP)
- РОП
- Руководитель группы продаж
- CRM‑администратор

## 3) Боль и JTBD
- Сделки передают между менеджерами, но новый ответственный не всегда делает следующий шаг.
- Руководитель не видит качество передачи и потери после смены владельца.

## 4) Измеримый эффект
- Снижение количества сделок без действия после передачи.
- Более прозрачная ответственность при перераспределении CRM.

## 5) УТП (позиционирование)
**“Покажем передачи сделок и проследим, что новый ответственный взял их в работу.”**

## 6) MVP (3–5 фич)
1. **Журнал смен ответственного по сделкам.**
2. **Список передач без нового действия за N часов/дней.**
3. **Фильтры по старому и новому ответственному.**
4. **Дайджест РОПу по рискованным передачам.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Сделки, ответственные, история изменений, активности, стадии.

### Триггеры и обновление
- Обработка события смены ответственного и периодическая проверка последующих действий.

### Права и безопасность
- Храним только факт смены, timestamps и ссылки на сделки; контент коммуникаций не читаем.

### Производительность
- Ограничение активными сделками и периодом последних передач.

## 8) Риски (и как снизить)
- **История изменений может быть неполной** → фиксировать события после установки.
- **Не вся передача требует действия** → настраиваемые исключения по стадиям.
- **Шумные алерты** → один дайджест вместо частых уведомлений.

## 9) Метрики успеха
- Количество передач без действия.
- Среднее время первого действия после передачи.
- Доля передач, закрытых в SLA.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “смена ответственного”
- “передача сделки”
- “контроль CRM”
- “ответственный CRM”
- “журнал сделок”

## 11) Упаковка для Маркета (черновик)
### Название
Контроль передачи сделок

### Буллеты ценности
- Журнал смен ответственного
- SLA первого действия после передачи
- Фильтры по менеджерам
- Дайджест руководителю
