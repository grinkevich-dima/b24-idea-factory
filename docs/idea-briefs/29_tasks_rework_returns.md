# Идея 29 — Задачи: Возвраты задач на доработку

## 1) Коротко (одна фраза)
Приложение считает возвраты задач на доработку и показывает, где команда теряет время на повторной работе.

## 2) Для кого (ICP)
- Руководитель проекта
- Тимлид
- Руководитель качества процесса

## 3) Боль и JTBD
- Задачи закрываются не с первого раза, но это не видно как отдельная метрика.
- Повторная работа съедает сроки и ресурсы.

## 4) Измеримый эффект
- Снижение количества повторных возвратов.
- Повышение качества постановки и выполнения задач.

## 5) УТП (позиционирование)
**“Покажем задачи, которые возвращаются на доработку, и причины потерь времени.”**

## 6) MVP (3–5 фич)
1. **Счётчик возвратов по задачам.**
2. **Отчёт по проектам, постановщикам и исполнителям.**
3. **Список задач с повторными возвратами.**
4. **Дайджест постановщику или руководителю.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Задачи, статусы, история изменений, постановщики, исполнители.

### Триггеры и обновление
- Фиксация переходов статусов после установки и ежедневный пересчёт отчёта.

### Права и безопасность
- Храним только статусы, timestamps и агрегаты возвратов; содержимое задач не копируем.

### Производительность
- Ограничение активными проектами и периодом анализа.

## 8) Риски (и как снизить)
- **В разных командах разные статусы доработки** → настройка переходов.
- **Нет явного статуса “доработка”** → можно считать возврат из “готово” в “в работе”.
- **Риск обвинительного отчёта** → позиционировать как улучшение процесса.

## 9) Метрики успеха
- Количество возвратов на задачу.
- Топ проектов по возвратам.
- Время, потерянное между возвратами и закрытием.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “возврат задач”
- “доработка задач”
- “качество выполнения задач”
- “повторная работа”
- “процесс задач”

## 11) Упаковка для Маркета (черновик)
### Название
Возвраты задач на доработку

### Буллеты ценности
- Считает повторную работу
- Показывает проблемные проекты
- Настраиваемые статусы возврата
- Дайджест руководителю
