# 08 — Просрочки по дедлайнам: радар — MVP + релиз + текст Маркета

## УТП

**Единый «радарный» экран просрочек по всем проектам и задачам: 3 зоны (горит / завтра / эта неделя) — мгновенная оценка масштаба проблем без открытия каждого проекта.**

## MVP (3–5 фич)

1. **Радарный дашборд** — 3 зоны с количеством задач: 🔴 просрочено, 🟡 дедлайн завтра, 🟠 дедлайн на этой неделе.
2. **Детализация** — клик по зоне → список задач с ответственным, проектом, сроком.
3. **Фильтры** — по проекту, ответственному, отделу.
4. **Алерт при критическом уровне** — уведомление руководителю, если в 🔴 зоне > N задач (настраивается).
5. **Настройки** — пороги для алерта, исключённые проекты, получатели уведомлений.

## Не-MVP (отложено)

- Тренды (график % просрочек по неделям)
- Прогноз просрочек (на основе скорости выполнения)
- Интеграция с Гантом
- Массовые действия (переназначить, продлить дедлайн)
- Drill-down по отделам/командам

## План релиза

### Неделя 1
- День 1–2: API-запросы к задачам + агрегация по зонам (просрочено / завтра / неделя)
- День 3–4: UI радара (3 зоны + числа + визуальная индикация)
- День 5: Детализация (список задач при клике на зону)

### Неделя 2
- День 1: Фильтры (проект, ответственный, отдел)
- День 2: Алерт при превышении порога
- День 3: Оптимизация производительности (кэширование, пагинация)
- День 4: Тестирование
- День 5: Публикация + тексты

| Установки | 400+ |

| Средняя оценка | ≥ 4.5 |
| Retention 30 дней | ≥ 60% |

## Риски и смягчения

| Риск | Смягчение |
|------|-----------|
| Производительность на больших порталах (10 000+ задач) | Фильтр: только активные задачи + кэш на 15 мин + пагинация |
| Шум (много просрочек при плохой дисциплине) | Топ-50 по важности + «скрыть старые просрочки» (> 30 дней) |
| Задачи без дедлайна | Отдельный счётчик «без дедлайна» + рекомендация поставить |

### Quick Start
1. Установите приложение
2. Откройте — видите радар с тремя зонами
3. Кликните на зону — видите задачи, которые требуют внимания
4. (Опционально) Настройте алерт при > N задач в красной зоне

### FAQ
- **Чем отличается от встроенного фильтра просроченных?** — Радар показывает ВСЕ проекты на одном экране + добавляет «предупреждающие» зоны (завтра, неделя), а не только уже просроченные.
- **Как работает с задачами без дедлайна?** — Отдельный счётчик. Рекомендация поставить дедлайн.
- **Тормозит на большом портале?** — Кэширование + показ топ-50 + фильтр по проектам.
