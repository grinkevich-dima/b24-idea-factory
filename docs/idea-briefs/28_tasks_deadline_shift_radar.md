# Идея 28 — Задачи: Задачи с частыми переносами дедлайна

## 1) Коротко (одна фраза)
Отчёт показывает задачи и проекты, где дедлайны переносили слишком часто, и помогает увидеть скрытые риски планирования.

## 2) Для кого (ICP)
- Руководитель проекта
- Тимлид
- Операционный руководитель

## 3) Боль и JTBD
- Дедлайн постоянно переносится, но задача формально не просрочена.
- Руководитель не видит системные переносы по проектам и исполнителям.

## 4) Измеримый эффект
- Снижение количества повторных переносов дедлайна.
- Более реалистичное планирование задач.

## 5) УТП (позиционирование)
**“Покажем задачи с частыми переносами дедлайна как ранний сигнал риска.”**

## 6) MVP (3–5 фич)
1. **Список задач с N переносами дедлайна.**
2. **Рейтинг проектов и исполнителей по переносам.**
3. **Порог уведомления после N переносов.**
4. **Детализация истории переносов.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Задачи, дедлайны, история изменений, ответственные, группы/проекты.

### Триггеры и обновление
- Фиксация событий изменения дедлайна после установки и периодический отчёт.

### Права и безопасность
- Храним только даты переносов и ссылки на задачи; комментарии не читаем.

### Производительность
- Считаем переносы за выбранный период и кэшируем агрегаты по проектам.

## 8) Риски (и как снизить)
- **История до установки недоступна** → честно показывать период наблюдения.
- **Легитимные переносы** → исключения по проектам и статусам.
- **Слишком много сигналов** → порог N переносов и топ‑список.

## 9) Метрики успеха
- Количество задач с повторными переносами.
- Среднее число переносов на задачу.
- Доля задач, стабилизированных после алерта.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “перенос дедлайна”
- “изменение срока задачи”
- “риск проекта”
- “контроль дедлайнов”
- “задачи Bitrix24”

## 11) Упаковка для Маркета (черновик)
### Название
Радар переносов дедлайна

### Буллеты ценности
- Видите скрытые риски до просрочки
- Рейтинг проектов по переносам
- Уведомление после N переносов
- История изменений сроков
