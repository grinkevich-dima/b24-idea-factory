# Матрица конкурентов (шаблон)

## Как искать
- 10–15 запросов: прямые, синонимы, смежные формулировки
- Категории: CRM / Аналитика / Задачи / Чаты (по контексту идеи)

## Таблица (заполнять 5–10 конкурентов)
Для каждого конкурента:
- **Название**
- **Ссылка (страница Маркета)**
- **Разработчик**
- **Установок / оценка**
- **Модель оплаты**: free / trial / подписка / разовая
- **Обещание (value prop)**: 1–2 предложения
- **Ключевые функции**: 3–7 буллетов
- **Какие объекты трогает**: CRM/Tasks/IM и какие сущности
- **Права (что запрашивает)**
- **Отзывы (сигналы)**: повторяющиеся боли/претензии/запросы
- **Слабые места**: 3–5 пунктов

## Итоги по рынку
- **Топ‑10 слабых мест** (повторяющиеся темы)
- **Белые пятна** (что никто не делает хорошо)
- **Возможности дифференциации** (2–3 фичи, которыми выигрываем)
