# Идея 17 — Аналитика: KPI РОПа “одна панель”

## 1) Коротко (одна фраза)
Единый дашборд для руководителя продаж с ключевыми показателями по лидам, сделкам, активности и просрочкам.

## 2) Для кого (ICP)
- РОП
- Коммерческий директор
- Собственник малого и среднего бизнеса

## 3) Боль и JTBD
- KPI продаж разбросаны по разным отчётам.
- РОП тратит время на ручной сбор цифр перед планёркой.

## 4) Измеримый эффект
- Сокращение времени подготовки к планёрке.
- Более быстрый контроль отклонений по менеджерам и воронкам.

## 5) УТП (позиционирование)
**“Одна панель РОПа показывает главные KPI продаж без ручного сведения отчётов.”**

## 6) MVP (3–5 фич)
1. **10–12 KPI**: новые лиды, конверсия, активные сделки, сумма в работе, просрочки, активности.
2. **Фильтры по периоду, менеджеру и воронке.**
3. **Сравнение с прошлым периодом.**
4. **Детализация по менеджерам.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Лиды, сделки, активности, ответственные, стадии, суммы.

### Триггеры и обновление
- Пересчёт агрегатов по расписанию и при открытии панели.

### Права и безопасность
- Используем CRM‑поля и агрегаты; не читаем содержимое коммуникаций.

### Производительность
- Периоды 7/30/90 дней по умолчанию и кэширование KPI.

## 8) Риски (и как снизить)
- **У компаний разные KPI** → настраиваемые виджеты в пределах MVP.
- **Большой объём CRM** → агрегаты по дням.
- **Споры по формулам** → показывать подсказки “как считается”.

## 9) Метрики успеха
- Время подготовки отчёта РОПа.
- Количество открытий панели в неделю.
- Доля KPI с положительной динамикой после внедрения.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “KPI РОПа”
- “дашборд продаж”
- “отчёт руководителя продаж”
- “CRM аналитика”
- “воронка продаж”

## 11) Упаковка для Маркета (черновик)
### Название
KPI РОПа: одна панель продаж

### Буллеты ценности
- Главные KPI продаж в одном месте
- Сравнение с прошлым периодом
- Детализация по менеджерам
- Быстрый старт без BI‑проекта
