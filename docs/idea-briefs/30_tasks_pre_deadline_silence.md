# Идея 30 — Задачи: Контроль задач без обновлений перед дедлайном

## 1) Коротко (одна фраза)
Приложение находит задачи, у которых дедлайн близко, но давно не было обновлений, и предупреждает ответственных до просрочки.

## 2) Для кого (ICP)
- Руководитель проекта
- Тимлид
- Операционный руководитель

## 3) Боль и JTBD
- До дедлайна остаётся мало времени, но по задаче тишина.
- Руководитель узнаёт о проблеме уже после просрочки.

## 4) Измеримый эффект
- Снижение количества внезапных просрочек.
- Рост доли задач, обновлённых до наступления дедлайна.

## 5) УТП (позиционирование)
**“Покажем задачи, которые молчат перед дедлайном, и напомним команде заранее.”**

## 6) MVP (3–5 фич)
1. **Список задач с дедлайном в ближайшие N дней и без обновлений M дней.**
2. **Настройки N/M по проектам.**
3. **Напоминание ответственному.**
4. **Дайджест руководителю по зонам риска.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Задачи, дедлайны, дата изменения, ответственные, группы/проекты.

### Триггеры и обновление
- Ежедневная проверка активных задач и ручное обновление списка.

### Права и безопасность
- Используем метаданные задач; не читаем комментарии и вложения.

### Производительность
- Фильтр по ближайшим дедлайнам резко ограничивает объём выборки.

## 8) Риски (и как снизить)
- **Задача может быть в работе без комментариев** → напоминание просит обновить статус, не обвиняет.
- **Много задач перед дедлайном** → группировка по ответственным.
- **Разные нормы по проектам** → настройки N/M.

## 9) Метрики успеха
- Количество задач без обновлений перед дедлайном.
- Доля задач, обновлённых после напоминания.
- Количество предотвращённых просрочек.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “задачи перед дедлайном”
- “нет обновлений”
- “предупреждение просрочек”
- “контроль задач”
- “дедлайн Bitrix24”

## 11) Упаковка для Маркета (черновик)
### Название
Задачи без обновлений перед дедлайном

### Буллеты ценности
- Ранний сигнал до просрочки
- Настройка окна дедлайна
- Напоминания ответственным
- Дайджест руководителю
