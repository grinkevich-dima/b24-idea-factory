# 10 — Дайджест для руководителя: анализ конкурентов

## Методология поиска

Ключевые запросы: «дайджест руководителя», «ежедневный отчёт чат», «автоматический отчёт», «сводка для руководителя», «daily digest», «утренний отчёт», «отчёт в чат», «статистика менеджеров чат», «что горит», «morning report bitrix24».

## Конкуренты

### 1. Ежедневный отчёт по сотрудникам в чат (PALAX)

| Параметр | Значение |
|----------|----------|
| Обещание | Автоматический отчёт в чат: задачи, звонки, продажи |
| Механика | Cron → сбор данных из Задач и CRM → форматированное сообщение в чат |
| Installs/оценка | 500+ установок, ~4.3 |
| Отзывы | «Много данных, но не видно что горит», «хочу фильтр по отделу», «нет приоритизации» |
| Запрашиваемые права | Полный доступ к чатам + задачам + CRM |
| Пробелы | Фокус на **статистике** (сколько сделано), а не на **проблемах** (что горит). Нет приоритизации. Избыточные права (чтение/запись чатов). Длинные сообщения. |

### 2. Рабочий авто-отчёт

| Параметр | Значение |
|----------|----------|
| Обещание | Отчёт по затраченному времени: задачи, звонки, встречи |
| Механика | Сбор данных по трудозатратам → отчёт руководителю |
| Installs/оценка | 300+ установок |
| Отзывы | «Полезно для учёта времени, но не для контроля проблем» |
| Запрашиваемые права | Задачи + дела |
| Пробелы | Фокус на **учёте времени**, а не на управленческих решениях. Не показывает просрочки и риски. |

### 3. Роботы CRM → уведомление в чат (встроенные)

| Параметр | Значение |
|----------|----------|
| Обещание | Отправка сообщений в чат при событиях в CRM |
| Механика | Робот «Отправить сообщение в групповой чат» на стадии |
| Installs/оценка | Встроено |
| Запрашиваемые права | — |
| Пробелы | Событийные (по одной сделке), а не агрегированные. Нет утреннего дайджеста. Нужно настраивать per-воронка. |

### 4. CRM-аналитика + email-отчёты (встроенные)

| Параметр | Значение |
|----------|----------|
| Обещание | Регулярные аналитические отчёты на email |
| Механика | Штатная CRM-аналитика → подписка на email |
| Installs/оценка | Встроено (тарифы Стандартный+) |
| Запрашиваемые права | — |
| Пробелы | Только email (не чат). Табличный формат (не краткий дайджест). Только CRM (нет задач). Нет «что горит». |

### 5. Супер отчёт руководителя (РОП)

| Параметр | Значение |
|----------|----------|
| Обещание | Комплексный отчёт для руководителя |
| Механика | Веб-интерфейс с дашбордом |
| Installs/оценка | 200+ установок |
| Отзывы | «Нужно заходить самому», «хочу получать в чат» |
| Запрашиваемые права | CRM |
| Пробелы | Пассивный (нужно зайти в приложение). Нет push-доставки. Нет кросс-модульности (CRM + задачи). |

## Топ слабых мест рынка

1. **Статистика вместо приоритетов** — конкуренты показывают «что сделано», а не «что горит / требует решения».
2. **Пассивная доставка** — нужно заходить в приложение или на email, а не получать в рабочий чат.
3. **Нет кросс-модульной сводки** — CRM отдельно, задачи отдельно. Руководителю нужна единая картина.
4. **Длинные нечитаемые отчёты** — 50 строк статистики вместо 5 строк «где горит».
5. **Избыточные права** — полный доступ к чатам (чтение/запись/удаление) для отправки одного сообщения.

## Что мы делаем лучше

- **Фокус на «что горит»**: дайджест = просроченные задачи + застрявшие сделки + задачи без активности. Только проблемы, только приоритеты.
- **Доставка в чат**: каждое утро в 09:00 → готовый к действию список.
- **Кросс-модульность**: задачи + сделки + дела — одно сообщение.
- **Краткость**: топ-10 проблем, не 50 строк статистики. Пустой дайджест = не отправляем.
- **Минимальные права**: чтение задач + чтение сделок + отправка сообщений (не полный доступ к чатам).
- **Настройка за 1 минуту**: выбрал отдел, настроил время — работает.
