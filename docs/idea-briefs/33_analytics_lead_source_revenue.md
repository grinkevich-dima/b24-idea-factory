# Идея 33 — Аналитика: Вклад источников лидов в выручку

## 1) Коротко (одна фраза)
Отчёт связывает источники лидов с выигранными сделками и показывает, какие каналы дают выручку, а не только заявки.

## 2) Для кого (ICP)
- РОП
- Маркетолог
- Коммерческий директор

## 3) Боль и JTBD
- Источники оценивают по количеству лидов, а не по выручке.
- Непонятно, какие каналы реально дают сделки и деньги.

## 4) Измеримый эффект
- Более точное распределение бюджета между источниками.
- Рост доли выручки из эффективных каналов.

## 5) УТП (позиционирование)
**“Покажем вклад источников лидов в сделки и выручку на одном отчёте.”**

## 6) MVP (3–5 фич)
1. **Отчёт источник → лиды → сделки → сумма выигрыша.**
2. **Конверсия по источникам и средний чек.**
3. **Фильтры по периоду, воронке и менеджеру.**
4. **Подсветка источников с низкой конверсией или высоким вкладом.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Лиды, сделки, источники, суммы, стадии выигрыша, даты создания и закрытия.

### Триггеры и обновление
- Ночной пересчёт агрегатов и ручное обновление отчёта.

### Права и безопасность
- Используем CRM‑поля и агрегаты; не читаем содержимое коммуникаций.

### Производительность
- Агрегаты по источникам и периодам, ограничение глубины истории по умолчанию.

## 8) Риски (и как снизить)
- **Плохое качество поля “Источник”** → показывать долю пустых/неизвестных источников.
- **Сложная атрибуция** → MVP использует первый/текущий источник из CRM без мультиканальности.
- **Длинный цикл сделки** → фильтры по датам лида и закрытия.

## 9) Метрики успеха
- Выручка по источникам.
- Конверсия лид → выигранная сделка.
- Доля сделок с заполненным источником.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “источники лидов”
- “выручка по каналам”
- “конверсия источников”
- “CRM маркетинг”
- “аналитика продаж”

## 11) Упаковка для Маркета (черновик)
### Название
Источники лидов: вклад в выручку

### Буллеты ценности
- Считает выручку по источникам
- Показывает конверсию и средний чек
- Подсвечивает слабые каналы
- Работает на стандартных CRM‑полях
