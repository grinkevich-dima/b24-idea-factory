# 07 — Зависшие задачи без изменений: анализ конкурентов

## Методология поиска

Ключевые запросы: «зависшие задачи», «stale tasks», «задачи без изменений», «контроль задач», «неактивные задачи», «задачи просрочены», «мониторинг задач», «задачи без активности», «болото задач», «forgotten tasks», «task monitoring».

## Конкуренты

### 1. Контроль необработанных задач в CRM (PINALL / Пинкит)

| Параметр | Значение |
|----------|----------|
| Обещание | Определить количество задач в сущностях CRM в нужном статусе |
| Механика | Подсчёт задач по статусам (закрыто/просрочено/в работе) привязанных к CRM-сущностям |
| Installs/оценка | ~200+ установок |
| Запрашиваемые права | Полный доступ к CRM + задачам |
| Пробелы | Фокус на CRM-задачах, а не на проектных. Нет критерия «дней без изменений». Нет дайджеста. Избыточные права. |

### 2. Встроенные счётчики и фильтры Bitrix24

| Параметр | Значение |
|----------|----------|
| Обещание | Подсветка просроченных задач и задач с изменениями |
| Механика | Счётчики в разделе «Задачи», фильтр по статусу, режим «Руковожу» |
| Installs/оценка | Встроено (бесплатно) |
| Запрашиваемые права | — |
| Пробелы | Нет фильтра «задачи без изменений N дней». Счётчики показывают только просроченные по дедлайну, а не по активности. Нет автоматического дайджеста. |

### 3. Роботы + триггеры (встроенные)

| Параметр | Значение |
|----------|----------|
| Обещание | Автоматическое перемещение задач по стадиям при изменении статуса/приближении срока |
| Механика | 4 триггера: изменение статуса, приближение срока, просрочка, изменение полей |
| Installs/оценка | Встроено (тариф Профессиональный+) |
| Запрашиваемые права | — |
| Пробелы | Триггеры работают только с задачами в проектах/Канбан (со стадиями). Нет триггера «нет изменений N дней». Нет агрегированного отчёта. |

### 4. BI-отчёт: Аналитика задач PRO

| Параметр | Значение |
|----------|----------|
| Обещание | Расширенная аналитика по задачам (статусы, сроки, нагрузка) |
| Механика | BI-отчёты с фильтрами, графиками, экспортом |
| Installs/оценка | ~100+ установок |
| Запрашиваемые права | Доступ к задачам |
| Пробелы | Отчёты — не проактивный инструмент. Нет «детектора тишины». Нужно самому зайти и проанализировать. |

### 5. Рабочий авто-отчёт

| Параметр | Значение |
|----------|----------|
| Обещание | Автоматический отчёт по затраченному времени в задачах |
| Механика | Сбор данных по времени + звонки + встречи → отчёт |
| Installs/оценка | ~300+ установок |
| Запрашиваемые права | Задачи + дела |
| Пробелы | Фокус на учёте времени, а не на «зависших» задачах. Не показывает задачи без активности. |

## Топ слабых мест рынка

1. **Нет критерия «дней без изменений»** — ни одно решение не определяет задачи, которые молчат N дней (не просрочены по дедлайну, а просто забыты).
2. **Нет проактивного дайджеста** — руководитель должен сам заходить в отчёт или фильтр.
3. **Встроенные инструменты ограничены** — триггеры работают только в Канбан, счётчики не показывают «тишину».
4. **Фокус на CRM, а не на проектах** — конкуренты привязаны к сделкам, а не к автономным задачам.
5. **Избыточные права** — конкуренты просят полный доступ к CRM для задач, не связанных с CRM.

## Что мы делаем лучше

- **Единственный критерий «дней молчания»**: задача без комментариев/смены статуса/вложений > N дней = зависшая.
- **Проактивный дайджест** в чат руководителю (ежедневно/еженедельно).
- **Работает для ВСЕХ задач** — не только CRM, не только Канбан.
- **Минимальные права**: только чтение задач + отправка уведомлений.
- **Мгновенный time-to-value**: установил → видишь список через 1 минуту.
