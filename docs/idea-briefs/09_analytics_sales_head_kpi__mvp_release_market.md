# 09 — KPI РОПа: одна панель — MVP + релиз + текст Маркета

## УТП

**10–12 ключевых метрик отдела продаж на одном экране — без настройки, без аналитика, без 50 вкладок. Установил → видишь всё за сегодня и вчера.**

## MVP (5 фич)

1. **Панель метрик** — 10–12 карточек на одном экране: новые лиды, новые сделки, закрытые (выигранные/проигранные), сумма выигранных, средний чек, конверсия лид→сделка, конверсия сделка→выигрыш, просроченные дела, звонки (вх/исх), среднее время до первого касания.
2. **Фильтры** — период (сегодня/вчера/неделя/месяц/произвольный), менеджер, воронка.
3. **Сравнение с прошлым периодом** — Δ% рядом с каждой метрикой (рост/падение).
4. **Детализация** — клик по метрике → список сущностей (сделки, лиды, дела).
5. **Алерт при аномалии** — если ключевая метрика (конверсия/звонки) упала > 20% от средней за неделю → уведомление в чат.

## Не-MVP (отложено)

- Plan/fact (требует ввода плановых значений — усложняет onboarding)
- Рейтинг менеджеров (геймификация)
- Экспорт в Excel/PDF
- TV-режим (дашборд на стену)
- Графики трендов за квартал
- Кастомные метрики

## План релиза

### Неделя 1
- День 1–2: API-запросы: лиды, сделки, дела, звонки + агрегация метрик
- День 3–4: UI панели (12 карточек + Δ% + цветовая индикация)
- День 5: Фильтры (период, менеджер, воронка)

### Неделя 2
- День 1: Детализация (drill-down по метрике)
- День 2: Сравнение с прошлым периодом
- День 3: Алерт при аномалии
- День 4: Оптимизация (кэширование агрегатов для больших порталов)
- День 5: Тестирование + публикация

## Метрики успеха

| Метрика | Цель (3 мес.) |
|---------|---------------|
| Установки | 500+ |
| Активные порталы (30 дней) | 200+ |
| Средняя оценка | ≥ 4.3 |
| Retention 30 дней | ≥ 55% |
| DAU/MAU | ≥ 40% (ежедневное использование) |

## Риски и смягчения

| Риск | Смягчение |
|------|-----------|
| Высокая конкуренция | Дифференциация через простоту: 0 настройки, 1 экран, ежедневный фокус |
| Производительность (1000+ сделок) | Кэш агрегатов на 15 мин + инкрементальное обновление |
| Разные воронки/типы сделок | Фильтр по воронке + дефолт «все воронки» |
| Точность метрик «звонки» | Зависит от подключённой телефонии — показывать «нет данных» если не настроена |

- ✅ Сравнение с прошлым периодом (Δ%) — сразу видно тренд
- ✅ Детализация в 1 клик — от числа к конкретным сделкам

### Quick Start
1. Установите приложение
2. Откройте — видите 12 метрик за сегодня
3. Переключите период или менеджера для детализации
4. (Опционально) Включите алерт при падении конверсии

### FAQ
- **Чем отличается от встроенной CRM-аналитики?** — Одна страница vs 10 разделов. Фокус на «сегодня/вчера» vs месячные отчёты. Звонки + дела + сделки в одном месте.
- **Нужно ли настраивать планы?** — Нет. MVP работает без планов — показывает факт + Δ. Планы — в будущей версии.
- **Работает с несколькими воронками?** — Да. Фильтр по воронке или «все сразу».
