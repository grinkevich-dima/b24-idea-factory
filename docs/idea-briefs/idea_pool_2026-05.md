# Пул идей (20+) — Bitrix24 Marketplace (CRM/Задачи/Аналитика/Чаты)

Формат: **боль → эффект → MVP‑ядро → сложность/риски**.

## CRM
1) **SLA реакции на лид (до первого касания)** → рост конверсии → дашборд+пороги+эскалации → средняя, спор “что считать касанием”.
2) **SLA по стадиям сделки (застревания)** → ускорение воронки → SLA по стадиям+список просрочек+алерты → средняя, шум уведомлений.
3) **Сделки без касаний N дней** → меньше “мертвых” сделок → список+фокус TOP‑10+быстрые действия → низко‑средняя, определение активности.
4) **Контроль заполненности карточек (Data Quality)** → чище CRM/отчёты → профили требований+скоринг+нарушения → средняя, вариативность полей.
5) **Причины проигрыша: контроль качества заполнения** → корректная аналитика → правила+отчёт по “пустым/мусорным” причинам → низко‑средняя.
6) **Скидки под контролем** → защита маржи → правила порогов+журнал отклонений → средняя, разные модели скидок.
7) **“Забытые счета/КП” в CRM** → меньше просрочек документооборота → список+алерты+эскалации → средняя, доступ к сущностям зависит от CRM.
8) **Потери из‑за просроченных дел по сделкам** → дисциплина → отчёт “сделки с просроченными делами”+напоминания → низкая.
9) **Качество полей телефона/email** → меньше возвратов/ошибок → валидатор+отчёт+массовое исправление (без интеграций) → средняя.
10) **Нарушения регламента по стадиям** (чек‑лист стадии) → снижение ошибок → чек‑лист+контроль выполнения → средняя, UX.

## Задачи/Проекты
11) **Риск проекта за 1 экран** → ранняя диагностика → риск‑скоринг+4 сигнала+алерт → средняя.
12) **Перегруз исполнителей (WIP)** → балансировка → отчёт+пороги+уведомления → низко‑средняя, спор метрик.
13) **Зависшие задачи без изменений N дней** → меньше “болота” → список+дайджест руководителю → низкая.
14) **Просрочки по дедлайнам: радар** → контроль сроков → дашборд+алерты+детализация → низко‑средняя.
15) **Задачи без дедлайна/ответственного** → качество управления → список нарушений+напоминания → низкая.
16) **План/факт часов по задачам (простая версия)** → контроль трудозатрат → отчёт+фильтры → средняя, зависит от дисциплины учёта времени.

## Аналитика/Отчёты
17) **KPI РОПа “одна панель”** → управляемость → 10–12 метрик+фильтры → средняя.
18) **Алерты по аномалиям (без ML)** → раннее предупреждение → правила Δ к прошлой неделе+уведомления → средняя, шум.
19) **Динамика конверсии по стадиям** → выявление провалов → отчёт+сравнение периодов → средняя.
20) **Качество CRM (ошибки данных)**: без суммы/кривые даты → меньше мусора → отчёт ошибок+ссылки на исправление → низко‑средняя.

## Чаты (осторожно по правам/масштабу)
21) **SLA ответа в чатах (молчание)** → скорость реакции → очередь “ждёт ответа”+эскалации → средне‑высокая, права/объём.
22) **Чек‑лист качества ответа** (оператор) → стандарты → шаблоны+оценка+отчёт → средняя, принятие.
23) **Дайджест для руководителя** (без ИИ): “что просрочено/где горит” → экономия времени → ежедневный отчёт в чат → низкая.

3. Сделки без касаний
