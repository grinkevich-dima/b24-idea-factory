# Идея 31 — Задачи: Еженедельный фокус исполнителя

## 1) Коротко (одна фраза)
Персональная панель и дайджест помогают исполнителю выбрать 5 главных задач недели по дедлайнам, статусам и риску просрочки.

## 2) Для кого (ICP)
- Руководитель команды
- Тимлид
- Исполнитель с большим количеством задач

## 3) Боль и JTBD
- У сотрудников много задач, и они теряют фокус на самом важном.
- Руководитель тратит время на ручное объяснение приоритетов недели.

## 4) Измеримый эффект
- Снижение распыления и просрочек по ключевым задачам.
- Более прозрачные недельные приоритеты.

## 5) УТП (позиционирование)
**“Каждый исполнитель видит свой короткий фокус недели без ручной сортировки задач.”**

## 6) MVP (3–5 фич)
1. **Панель “5 задач недели” для исполнителя.**
2. **Правила приоритизации**: дедлайн, просрочка, статус, проект.
3. **Еженедельный дайджест в понедельник.**
4. **Настройки исключений и лимита задач.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Задачи, дедлайны, статусы, ответственные, проекты/группы.

### Триггеры и обновление
- Еженедельная генерация фокуса и ручное обновление панели.

### Права и безопасность
- Показываем только задачи, доступные пользователю; не храним лишний контент.

### Производительность
- Расчёт по задачам конкретного исполнителя и ближайшему периоду.

## 8) Риски (и как снизить)
- **Спор правил приоритизации** → понятные настройки и объяснение “почему задача в фокусе”.
- **Дублирование стандартного списка задач** → ценность в коротком топ‑5.
- **У разных ролей разные приоритеты** → пресеты.

## 9) Метрики успеха
- Доля задач из фокуса, закрытых за неделю.
- Количество просрочек по задачам в фокусе.
- Открытия персонального дайджеста.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “фокус недели”
- “приоритеты задач”
- “личный дайджест задач”
- “топ задач”
- “план недели”

## 11) Упаковка для Маркета (черновик)
### Название
Еженедельный фокус исполнителя

### Буллеты ценности
- Топ‑5 задач недели
- Понятные правила приоритета
- Персональный дайджест
- Меньше распыления
