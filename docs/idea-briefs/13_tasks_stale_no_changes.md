# Идея 13 — Задачи: Зависшие задачи без изменений N дней

## 1) Коротко (одна фраза)
Приложение находит задачи, в которых давно не было изменений, и помогает руководителю быстро вернуть их в работу.

## 2) Для кого (ICP)
- Руководитель проекта
- Тимлид
- Руководитель операционной команды

## 3) Боль и JTBD
- Задачи формально открыты, но по ним давно нет движения.
- Стандартные списки не всегда показывают, что именно зависло и сколько дней молчит.

## 4) Измеримый эффект
- Сокращение количества задач без активности больше N дней.
- Ускорение среднего цикла выполнения задач.

## 5) УТП (позиционирование)
**“За минуту покажем задачи без движения и отправим руководителю понятный дайджест.”**

## 6) MVP (3–5 фич)
1. **Список зависших задач с количеством дней без изменений.**
2. **Настройка порога N дней и исключений по статусам.**
3. **Фильтры по проекту, ответственному и постановщику.**
4. **Дайджест руководителю раз в неделю или день.**

## 7) Реализуемость и тех‑чеклист (10 минут)
### Данные и объекты
- Задачи, статусы, ответственные, постановщики, дата последнего изменения.

### Триггеры и обновление
- Периодическая проверка активных задач; ручное обновление на экране списка.

### Права и безопасность
- Не читаем тексты комментариев; используем даты, статусы и ссылки на задачи.

### Производительность
- Берём только незавершённые задачи и ограничиваем период создания/изменения.

## 8) Риски (и как снизить)
- **Непонятно, что считать активностью** → в MVP используем дату изменения задачи.
- **Много старых задач при первом запуске** → показывать топ‑50 и фильтры.
- **Шум для отложенных задач** → исключения по статусам и проектам.

## 9) Метрики успеха
- Количество зависших задач.
- Средний возраст задач без изменений.
- Доля задач, обновлённых после дайджеста.

## 10) Поиск конкурентов (ключевые запросы для Маркета)
- “зависшие задачи”
- “задачи без изменений”
- “stale tasks”
- “контроль задач”
- “дайджест задач”

## 11) Упаковка для Маркета (черновик)
### Название
Зависшие задачи: детектор без движения

### Буллеты ценности
- Список задач без активности
- Порог N дней и исключения
- Дайджест руководителю
- Быстрые ссылки на исправление
